# JISMemo 개발일지

## 버전 1.3 (2025년 1월)

### 주요 개선사항

#### 🔧 텍스트/이미지 붙여넣기 기능 개선
- 클립보드에서 텍스트와 이미지를 더욱 안정적으로 붙여넣기 가능
- 다양한 이미지 포맷 지원 확대
- 붙여넣기 성능 최적화

#### 🖼️ 여러 이미지 지원 및 개별 삭제
- 하나의 메모에 여러 개의 이미지 첨부 가능
- 각 이미지별 개별 삭제 기능 추가
- 이미지 순서 변경 기능 구현

#### 🎨 UI 크기 및 폰트 개선
- 사용자 친화적인 인터페이스 크기 조정
- 가독성 향상을 위한 폰트 최적화
- 고해상도 디스플레이 지원 개선

#### 📜 스크롤바 추가
- 긴 메모 내용에 대한 스크롤 기능 개선
- 커스텀 스크롤바 디자인 적용
- 스크롤 성능 최적화

#### 🌐 기본 언어 영문 설정
- 프로그램 시작 시 기본 언어를 영문으로 변경
- 다국어 지원 기능 강화
- 언어 전환 기능 개선

#### 🐛 각종 버그 수정
- 메모 저장 시 발생하던 간헐적 오류 수정
- 이미지 로딩 관련 메모리 누수 문제 해결
- 프로그램 종료 시 안정성 개선
- 특수 문자 입력 시 발생하던 문제 수정
